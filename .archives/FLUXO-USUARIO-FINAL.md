### Passo 4️⃣: Usar o Sistema

#### 📊 Dashboard
- Vendas do dia/mês
- Clientes
- Estoque
- Financeiro

#### 💰 PDV (Ponto de Venda)
- Vendedor registra venda
- Busca cliente
- Registra itens vendidos
- Fecha venda

#### 📋 Gestão
- Cadastrar clientes
- Controlar estoque
- Ver relatórios financeiros
- Emitir NFC-e (nota fiscal)

#### 👤 Minha Conta
- Alterar senha
- Exportar dados (LGPD)
- Deletar conta (se quiser sair)

## 📊 Dados Coletados (Transparência)

O que armazenamos:
- Email, senha (criptografada), nome
- Dados da loja (nome, telefone, endereço)
- Vendas (data, itens, cliente, valor)
- Clientes (nome, email, telefone, CPF/CNPJ — opcional)
- Estoque (produtos, quantidade, preço)

**O que NÃO coletamos:**
- ❌ Dados pessoais além do necessário
- ❌ Histórico de navegação
- ❌ Localização (sem permissão)
- ❌ Dados de redes sociais

**LGPD Compliance:** Lojista pode deletar conta e todos dados são removidos.
